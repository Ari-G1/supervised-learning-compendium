# Supervised Learning Compendium

Final project for Introduction to Artificial Intelligence.

## Repository Structure

- `part1_regression/` - Regression task using the Auto MPG dataset
- `ClassificationTaskCIFAR10.py` - Classification task using CIFAR-10
- `figures/` — Figures generated by the classification script
- `PyTorchNN.ipynb` - Neural network classification using PyTorch

## Datasets

- **Auto MPG**: Publicly available at https://archive.ics.uci.edu/dataset/9/auto+mpg
- **CIFAR-10**: Publicly available at https://www.cs.toronto.edu/~kriz/cifar.html  
  Download the Python version and place it in `part2_classification/cifar-10-batches-py/`

## Notes

- Saved model files are not included in this repository as they can be reproduced by running the scripts.
- All results reported in the Overleaf report are reproducible by running the provided code.
